# Portfolio - Amina Bećirević

Moj lični portfolio izrađen u **HTML** i **CSS**, prikazuje moju edukaciju, vještine i projekte razvijene tokom studija i studentske prakse.

## Struktura portfolija

- **Navigation:** brzo prebacivanje između sekcija About, Skills, Portfolio i Contacts  
- **About me:** kratak opis obrazovanja, iskustva i interesa u IT-u  
- **Skills:** frontend i programske tehnologije koje poznajem
- **Portfolio:** prikaz dva glavna projekata sa linkovima na GitHub repozitorije  
  - **Frontend project:** Frontend web projekat korištenjem HTML/CSS i osnova JS
  - **Web application:** full-stack web aplikacija s frontendom u HTML/CSS i backendom u C# povezanim na SQL Server bazu podataka
  - **Webshop Database:** SQL Server baza podataka kreirana tokom studentske prakse u BH Telecomu    
- **Contacts:** e-mail, telefon i LinkedIn profil

## Tehnologije

- HTML5, CSS3, Responsive design  
- JavaScript (osnovno znanje)  
- SQL Server Management Studio (SSMS)  
- C# za backend u projektima s bazom podataka  

## Kako koristiti

1. Klonirajte repozitorij:  
   ```bash
   git clone https://github.com/aminabecirevic/Portfolio.git
   ```
2. Otvorite `index.html` u pregledniku da pregledate portfolio lokalno.  
3. Sve dodatne datoteke (CSS, slike, dokumenti) nalaze se u istom folderu ili podfolderima.
